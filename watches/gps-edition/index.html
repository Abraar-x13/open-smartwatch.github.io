GPS Edition

Specifications

  • ESP32-pico-D4: 2x240MHz, 320KB RAM
  • Bluetooth 4.2 BR/EDR BLE
  • WiFi 2.4GHz 802.11 b/g/n
  • GC9A01 240x240 16bit TFT display (round)
  • BMA400 Accelerometer + Pedometer
  • MCP73831 LiPo Charger
  • CH340E USB Serial
  • Quectel L96 GPS module
  • 4MB RAM
  • microSD
