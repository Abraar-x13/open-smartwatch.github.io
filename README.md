## Goals

The goal is to build an open source smartwatch, with step counting, GPS tracking and maps.

- ESP32 (Arduino programmable)
- GPS
- Time
- Sensors (Acceleromter, Step Counting)
- Li-Ion Battery
- USB Serial
- uSD Card (to provide open streetmap tiles)
